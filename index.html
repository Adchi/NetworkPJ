Lesson 1 Introduction of Networking

Interested in learning about networking? Let Network Direction help you get started.
This video is for people that are first starting out in networking. All you need to bring is interest and enthusiasm.
My goal of the network Fundamental Series is to help anyone wanting to get into the IT field learn more about Networking.
We will introduce you to the terminology used in the field and help you understand what it means. For instance, have you heard of protocols? We're going to see what these are, and use the #Ethernet protocol as an example.

Lesson 2 TCP/IP

Did you know that IP addresses are two addresses in one? Yes, it's true! An IP includes the host address, as well as the address of the network it resides in.
Speaking of networks, we have changed how we address them over time. One of the early methods was to use classes. Perhaps you've heard of class A, B, and C networks?
But this has its limitations. So, we also have classless networks, or CIDR (Classless Inter-Domain Routing). This introduces a new concept: The subnet mask. Now we can break up networks as we see fit!
